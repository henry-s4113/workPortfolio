CS 133JS, Lab 1, Part 1

This part of your lab assignment is to run the examples in You Don't Know JS: Up and Going, Ch. 1 in a browser console, then copy them to this HTML file and run them again there. Instructions:
  1. Copy the code from the book to the console and run it. Experiment with it by changing the code so it produces different results, but still uses the same main JS feature.
  2. Change the calls that produce output from console.log() to document.write() so that the Output goes to the web page instead of the console.
  3. Copy and paste the code into the script element in the source code for this web page. Replace the comment // put your code here with your code.
